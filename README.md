# html-css-js-ex

🎈 **SVG 互動球形展示 DEMO**

此專案示範使用 HTML / CSS / JavaScript 建立一個可互動的圓形動畫物件，可拖曳、反彈、點擊變色。

## ✅ Demo 功能

- SVG
- 球形動畫反彈效果
- 拖曳控制球位置
- 點擊變色與速度加快
- 類光源設定
- 球形放大縮小
- Rainbow Gradient
- Shapes with Image
- Periodic Table of the Elements
- Chromium Architecture List

## 📄 GitHub Pages 設定步驟

1. 將專案 push 到 GitHub Repo
2. 開啟 repo 的 `Settings → Pages`
3. 設定：
   - **Source:** `main` branch
   - **Folder:** `/ (root)`
4. 等待 GitHub Actions 執行完畢，自動部署成功
