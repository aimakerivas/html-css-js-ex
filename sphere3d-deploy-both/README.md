# sphere3d-deploy-both

## ✨ 本專案支援兩種 GitHub Pages 部署方式：

---

### ✅ 方法一：Deploy from a Branch

1. 將 `/docs` 作為部署來源
2. 設定 Pages：
   - Source: `Deploy from a branch`
   - Branch: `main` / `/docs`
3. 適用靜態內容快速展示

---

### ✅ 方法二：GitHub Actions 自動部署

- 預設已含 `.github/workflows/gh-pages.yml`
- 推送後自動上傳並發佈 GitHub Pages
- 可支援 CI/CD、自動轉檔等流程

---

🔗 `docs/index.html` 為首頁
